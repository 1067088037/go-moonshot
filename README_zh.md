## Go Moonshot

**简体中文** | [English](README_en.md)

This library provides unofficial Go Clients for [Kimi](https://kimi.moonshot.cn) which created by [MoonshotAI](https://moonshot.cn).

## Feature

- 人体工学的API，链式操作
- 完全的API支持
- 预定义枚举
